WEB / INTERNET TERMINOLOGIES

Here are some common web/internet terminologies:

  1. WWW (World Wide Web): a system of interlinked hypertext documents accessed via the internet.
  2. Internet: a global network of interconnected computers and servers that communicate with each other using standardized protocols.
  3. Web Browser: a software application that allows users to access and view web pages, e.g., Google Chrome, Mozilla Firefox.
  4. Web Server: a computer that hosts websites and serves web pages to users, e.g., Apache, IIS.
  5. Domain Name: a unique name that identifies a website, e.g., (link unavailable), (link unavailable).
  6. URL (Uniform Resource Locator): a web address that specifies the location of a web page, e.g., (link unavailable).
  7. HTTP (Hypertext Transfer Protocol): a protocol that governs the communication between web browsers and web servers.
  8. HTML (Hypertext Markup Language): a markup language used to create web pages, e.g., headings, paragraphs, images.
  9. CSS (Cascading Style Sheets): a styling language used to control the layout and appearance of web pages.
  10. JavaScript: a programming language used to add interactivity to web pages, e.g., animations, form validation.
  11. IP Address (Internet Protocol Address): a unique numerical address assigned to each device on a network, e.g., 192.168.1.1.
  12. ISP (Internet Service Provider): a company that provides internet access to customers, e.g., Verizon, AT&T.

Examples:

  • A user accesses a website using a web browser, which sends an HTTP request to the web server.
  • A web developer uses HTML, CSS, and JavaScript to create a web page.

Tips for Working with Web/Internet Terminologies:

  • Understand the basics of web development, including HTML, CSS, and JavaScript.
  • Familiarize yourself with web browsers and web servers.
  • Learn about internet protocols, such as HTTP and IP addresses.
